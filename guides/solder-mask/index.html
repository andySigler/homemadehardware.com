Solder Mask

Solder mask is a colorful layer that protects the copper on your PCB from the environment, and makes soldering much easier.

  1. Outline

Outline

Fab Time: 20-60 minutes (per layer)

Design

  1. Add copper color to design beforehand
    • Make sure the copper looks good
  2. Copper parts will always shrink some amount
    • UV leakage around edge of transparency
    • Why timing is so important

Prep

  1. Always measure UV timing beforehand (on each color)
    • Create template image
  2. Start with dark, end with light colors
    • Because when cleaning, dark colors might ruin lighter ones

Cure

  1. Tape printer paper down to table
  2. Create cellophane "frame" around the board
    • Cutout larger cellophane for entire board
    • Add liquid solder mask
    • Add larger cellophane layer
    • Add transparency design
    • Add glass cover
  3. Cure with UV light
    • No reflective surfaces to avoid sideways light
    • Let sit for a few minutes after cure
  4. Use a “roller” (pen diameter) when removing cellophane

Clean

  1. Mask goo is only cleanable with alcohol (not soap/water)
  2. Remove excess mask goo first
    • Try and keep away from rest of mask
  3. Then, remove easy goo from all copper parts
  4. Finally, gently clean entire board
  5. Cut away excess mask with a blade (if you must)
