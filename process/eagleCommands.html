Common Commands

Move

Move a part or connection around

Rotate

Click a part to rotate it 90 degrees

Change

This command can do anything, and once selected, will apply to anything you click. For example, if I wanted to chagne the width of all my routes, I would say "change width", then select the width I want, and now every route I click will change to that width.

Text

Write some text, NOT attached to any part of connection

Group

Commands can act on multiple parts/connections at once. Here is how you group a command:

  1. Enter the command you would like to group (for example: "move")

  2. Enter the group command (type "group")

  3. Click and drag to highlight the parts/connections, and release

  4. Right-click one of the highlighted parts, and a menu will appear next to your mouse

  5. Click your grouped command (for example: "Move: Group")

Schematic Commands

Add

Load all libraries, where you can pick a new part to add

Delete

Click a part or a connection to delete

Net

Draw electrical connection between two pins

Copy

Copy a part or connection by clicking it

Name

Give a part or connection a name

Label

Display the "name" of a part or connection (most useful for connections)

Board Commands

Route

Draw the traces that will become phyical copper traces. Turn those thing yellow "unrouted" lines to real traces, either on the Top (red) or Bottom (blue) layer.

Ripup

Remove a trace that was drawn with the "route" command. This does not delete a connection, but only removes the trace you drew for it.
